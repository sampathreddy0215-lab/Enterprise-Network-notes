# BGP Basics

## What is BGP?
Border Gateway Protocol (BGP) is the routing protocol used to exchange routing information between autonomous systems on the internet.

## Types of BGP
- eBGP (External BGP)
- iBGP (Internal BGP)

## Important Attributes
- AS Path
- Local Preference
- MED
- Next Hop

## Common Uses
- ISP connectivity
- Multi-homing
- Traffic engineering
- SD-WAN underlay routing
